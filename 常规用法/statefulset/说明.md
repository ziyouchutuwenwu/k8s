## 说明

1.20 以后，nfs 的 provisioner 镜像不支持 selfLink，需要换 provisioner 的 image

## pv 无法删除处理

```sh
kubectl get pv -n test
```

```sh
kubectl patch pv xxxxxxx -n test -p '{"metadata":{"finalizers":null}}'
```

查看

```sh
kubectl describe pv xxxxxx -n test | grep Finalizers
```
