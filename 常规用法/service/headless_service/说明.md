# headless_service

## 测试

### deployment 测试

```bash
kubectl create -f ./nginx-deploy.yaml
kubectl create -f ./headless-service.yaml
```

查看 service 对应的 endpoints

```sh
kubectl describe service ddd
```

看下 deployments 对应的 ip

```sh
kubectl get pod -o wide
```

登入 pod,查看 dns 记录

```bash
kubectl exec -it busybox -- /bin/sh
nslookup ddd
```

可以看到三个 ip
